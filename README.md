## 📌 Sobre el proyecto

Juan Pérez, un profesional independiente, lleva sus finanzas anotadas en papel y notas dispersas, lo que le impide tener una visión clara de su situación mensual. **My Budget Master** nace para resolver ese problema: un sistema simple e intuitivo donde cualquier persona, sin conocimientos contables, pueda registrar sus movimientos y tomar mejores decisiones sobre su dinero.

Un poco más sobre el proyecto:

* 🎓 Desarrollado como proyecto final del curso **Programación Orientada a Objetos (POO)**, dictado en la **FIIS - UNI**
* 💵 Registro de **ingresos y gastos** con fecha, monto, categoría y descripción
* 🗂️ Gestión completa de **categorías** personalizadas (crear, editar, eliminar)
* 📈 **Reportes** mensuales y trimestrales, con desglose por categoría en tablas y gráficos
* 🚨 **Presupuestos** por categoría con alertas automáticas al excederse el gasto previsto
* 🔐 Autenticación de usuario para proteger la información financiera
* 📱 Diseño pensado para uso constante y flujos simples de registro

### ⚙️ Backend — API REST (`/backend`)

| Tecnología | Uso en el proyecto |
|---|---|
| ☕ **Java 21** | Lenguaje base del backend |
| 🍃 **Spring Boot** | Framework principal de la API REST (controladores, servicios, DTOs) |
| 🔗 **Spring Data JDBC** | Acceso y persistencia de datos (transacciones, categorías, presupuestos) |
| 🔒 **Spring Security Crypto** | Cifrado de credenciales de usuario |
| 🗄️ **SQL Server** | Motor de base de datos relacional |
| 📦 **Maven** | Gestión de dependencias y build |
| 🧩 **Lombok** | Reducción de código boilerplate en entidades y DTOs |

### 🎨 Frontend — SPA (`/frontend`)

| Tecnología | Uso en el proyecto |
|---|---|
| ⚛️ **React 18 + TypeScript** | Interfaz de usuario y tipado estático |
| ⚡ **Vite** | Bundler y entorno de desarrollo |
| 💨 **Tailwind CSS + shadcn/ui (Radix UI)** | Sistema de diseño y componentes accesibles |
| 🧭 **React Router** | Navegación entre Dashboard, Transacciones, Presupuestos, Reportes, etc. |
| 📋 **React Hook Form + Zod** | Formularios y validación de datos |
| 🔄 **TanStack Query** | Manejo de estado remoto y peticiones a la API |
| 📊 **Recharts** | Visualización gráfica de reportes financieros |
